# Product Engineer (Frontend) 상시 채용

**회사**: 플렉스
**경력**: 0년
**지역**: 서울
**카테고리**: 디자인, IT_개발 / 프론트엔드, UXUI_프로덕트
**마감**: 확인 못함

## 요약 (zighang summary)

### 담당업무(Frontend)

- People Platform 개발(인사관리, 시간 관리, 급여 자동화, 전자 계약, 평가 및 보상, 채용 포함)
- 사용자 경험 책임 및 프런트엔드 기술 전문성 확보
- 문제 정의부터 개발-테스트-배포-운영까지 서비스 생애 주기 참여
- 복잡한 비즈니스 로직의 이해 쉬운 표현과 재사용 가능 구조 설계
- 플렉스팀의 기술 수준 및 개발 문화 성장 지원


### 담당업무(Design System)

- 디자인 시스템 fx 개발 및 관리
- UX/UI 패턴 설계 및 재사용 가능한 추상화 제공
- 일관된 UX/UI 경험 및 속도 유지에 기여


### 필수 기술 스택

- 코어: Node.js, TypeScript, React, Next.js
- 상태 관리: React Context, React Query, React Hook Form
- 스타일링: fx, Radix UI, Stitches
- 빌드: webpack, swc, Rollup
- 테스트: Jest, Testing Library, Storybook
- CI/CD: GitHub Actions, Jenkins, Kubernetes, Argo CD, Turborepo
- 로깅/모니터링: Sentry, Kibana, Grafana, Splunk, Prometheus


### 자격요건

- 프런트엔드 영역 및 웹 서비스 환경에 대한 깊은 이해
- 고객 문제를 유연한 시스템으로 해결하는 역량
- 기술 부채 인지 및 타협 없는 해결 주도 경험
- 자동화된 단위/통합 테스트 작성·실행 경험
- 서비스 생애 주기 전반의 흐름 이해 및 문제 해결 능력
- 조직 생산성 향상을 위한 문화/프로세스 구축 경험
- 디자인 시스템 이해를 바탕으로 유연하고 확장 가능한 컴포넌트 설계 능력
- HTML, CSS, JS, React에 대한 깊은 이해를 토대로 유려한 UI/UX 구현

- 엔지니어링적으로 주도적으로 선택했던 경험
