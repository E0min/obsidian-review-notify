### 소개

콘센트릭스는 미국 나스닥에 상장된 매출 13조, 직원수 44만명, 전 세계 70여개 국가에서 사업을 영위하는 글로벌 기업 콘센트릭스의 한국 법인으로 디지털 마케팅 및 이커머스 컨설팅 전문 기업입니다.

콘센트릭스는 디지털 마케팅 전 영역(전략컨설팅, UXUI, 웹컨텐츠, 구축/운영, 분석)에 대한 END TO END 서비스 제공 및 E-commerce 플랫폼의 기획,구축,운영을 수행하며 글로벌 Top-tier 대기업들과의 전략적 파트너로서 입지를 공고히 다져오고 있습니다.

뷰티/코스메틱/전자/자동차 등 글로벌 기업들과의 협업을 통해 다양한 업무 경험을 쌓고 역량을 강화하며 콘센트릭스와 함께 수평적이고 수직적인 성장을 원하신다면 적극적인 지원을 부탁드립니다.



### 주요업무

역할: 멀티 에이전트 오케스트레이션 및 AI 서비스 로직 구현

포지션 개요

LLM/Agent Engineer는 자연어 기반 업무 요청을 분석하고, 질의 유형 분류, Agent 라우팅, Tool-calling, 상태 관리, 응답 생성 로직을 구현하는 역할입니다. LangGraph 등의 orchestration tool을 이용하여 Agent Workflow를 설계 및 개발하며, 엔터프라이즈 업무 프로세스를 AI 서비스로 전환합니다.

• Multi-Agent Workflow 설계 및 구현
• 요청, 질의, 비교, 리포트 등 업무별 Agent 로직 구현 FastAPI
• FastAPI 기반 AI 서비스 API 개발LLM AP
• LLM API 비용, 응답 지연, 모델 라우팅 최적화
• 내부 도메인 전문가와 협업하여 업무 규칙을 Agent 로직으로 전환
