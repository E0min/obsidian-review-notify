# [플랫폼본부] 메이플시너지실 프론트엔드 개발자 (계약직)

**회사**: 넥슨
**경력**: 3년~100년
**지역**: 경기
**카테고리**: IT_개발 / 프론트엔드
**마감**: 확인 못함

## 요약 (zighang summary)

### 조직 소개

- 넥슨 게임 데이터 및 AI 기술 기반 솔루션 설계 및 구현
- 게임 사용자 몰입감 향상 및 구성원 업무 효율성 증대 목표
- 메이플스토리 프랜차이즈 웹, 모바일 앱, UI/UX, AI 솔루션 통합 지원
- 프론트엔드, 백엔드, 모바일 등 폭넓은 영역의 개발 경험 제공


### 주요업무

- 기획, 디자인, 백엔드 협업을 통한 웹 서비스 UI/UX 구현
- React 및 레거시 환경 코드 분석 및 리팩토링
- 서비스 품질 및 성능 개선
- RESTful API 연동 및 클라이언트 상태 관리
- 크로스 브라우징 및 반응형 웹 표준 준수 지원


### 지원자격

- 3년 이상의 프론트엔드 개발 실무 경험
- HTML, CSS, JavaScript(ES6+) 숙련도
- Next.js 프로젝트 경험
- Git 활용 팀 협업 및 버전 관리 능력


### 우대사항

- TypeScript 활용 프로젝트 경험
- Figma 등 디자인 툴 기반 협업 능력
- 대규모 트래픽 서비스 개발 및 운영 경험
- Nuxt.js 활용 경험


### 포트폴리오 제출 안내

- 프로젝트 포트폴리오(PDF, Notion 등), GitHub 링크, 서비스 데모 페이지 중 선택 제출
- 프로젝트 개요 및 목적 포함
- 본인 기여도(기능, 코드, 역할 등) 명시
- 사용 기술 스택 및 문제 해결 과정, 성과 포함


### 전형절차

- 서류전형
- 면접전형
- 최종합격
