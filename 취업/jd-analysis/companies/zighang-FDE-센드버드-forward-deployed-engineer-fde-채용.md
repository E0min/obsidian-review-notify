# Forward Deployed Engineer (FDE) 채용

**회사**: 센드버드
**경력**: 2년~3년
**지역**: 서울
**카테고리**: IT_개발 / 웹풀스택
**마감**: 확인 못함

## 요약 (zighang summary)

### 담당업무

- 고객 맞춤형 AI 에이전트 구축, 통합 및 배포
- 고객과 직접 협업하며 Sendbird AI Agent 플랫폼의 기술적 기반 설계 및 구현
- 시스템 프롬프트 튜닝, 툴체인 구성, 에이전트 행동 반복 개선
- LLM API(OpenAI, Claude 등) 활용 에이전트 프로토타입 및 배포
- Next.js/React 기반 인터페이스 및 대시보드 개발
- RAG 파이프라인/지식베이스/커스텀 툴 백엔드 API 통합
- 고객 배포 기술 구현 및 모니터링
- 기술 문서화 및 데모 리딩


### 자격요건

- Python 등 핵심 언어에 대한 이해와 코딩 능력 보유
- React 및 최신 자바스크립트/타입스크립트 기반 프론트엔드 개발 역량
- Node.js, Django 등 백엔드 프레임워크를 활용한 풀스택 개발 능력
- 에이전트 프레임워크(LangChain 등) 또는 LLM 기반 툴체인 경험 보유
- RAG 파이프라인 및 지식베이스 연동 경험 습득 또는 의지가 강한 자
- CI/CD 및 관측성(로그, 메트릭 대시보드, 가동시간 모니터링) 실무 지식
- 현대 클라우드 플랫폼(W AWS Lambda, Vercel, GCP Cloud Functions 등) 배포 경험
- 애자일(크로스펑셔널) 팀과의 협업 능력
- 모호한 상황에서도 주도적으로 진행하고, 빠르게 학습하며 책임감이 강한 자


### 우대사항

- 실시간 에이전트 성능 튜닝 또는 라이브 배포 디버깅 경험
- 0-1 환경에서의 업무 경험, 고객 대상 구현 또는 기술 컨설팅 경험


### 근무환경

- 유연근무제 운영
- 사무실 3일 이상 팀 합류를 기본 원칙으로 하는 현장 협업 문화


### 복리후생

- 스톡옵션(1년 이후 행사 가능)
- 언어학습 지원비 월 지급
- 직무 개발, 체력 단련, 도서 구입, 건강 관리 등 다양한 본인 성장 지원 금액 연간 제공
- Seolleung Office 주차 지원 또는 택시 이용 지원
- 본인, 배우자, 자녀를 포함한 그룹보험 지원
- 정기 건강검진 지원
- 필수 연차에 추가로 7일의 추가 휴가 제공
- 출산휴가 12주 전액 급여 보장
