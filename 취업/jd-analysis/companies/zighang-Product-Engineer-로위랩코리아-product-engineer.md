### 소개

부동산 개발사와 세일즈 에이전시를 위한 세일즈 퍼널을 설계하고, 사용자 문제를 정의해 데이터 기반으로 전환 성과를 만들어내는 SaaS 제품을 만듭니다.

로위랩은 3D 공간 탐색 경험을 단순한 콘텐츠에 머무르게 하지 않고, 실제 세일즈 전환으로 이어지는 데이터 구조로 확장하고 있습니다. 고객은 링크 하나로 공간을 탐색하며, 이 과정에서 발생하는 행동 데이터는 세일즈 의사결정과 전환으로 연결됩니다.

이 포지션은 랜딩부터 탐색, 행동, 전환까지 이어지는 전 과정을 설계하고, 전환율과 사용자 행동 데이터를 기반으로 성과를 개선하는 역할을 담당합니다. 제품 초기 단계에서 구조를 직접 설계하고, 실제 사용과 매출로 이어지는 제품 성장에 기여할 수 있습니다.



### 주요업무

· React / Next.js 기반 SaaS 웹 서비스 설계 및 개발
· 사용자 흐름 및 세일즈 퍼널 구조 설계
· 사용자 행동 데이터를 기반으로 기능 및 서비스 개선
· REST API 기반 백엔드 연동 및 프론트 구조 설계
· 공통 컴포넌트 및 재사용 가능한 구조 설계
· PRD, 유저 플로우 등 서비스 기획 문서 작성
· 기획/디자인/개발 조직과 협업하여 제품 개선 및 기능 구현
· 개발/디자인 조직과 협업하여 실제 구현 가능한 기획 수행
