# Frontend Engineer ( Capora 물류 시스템 )

**회사**: 42dot
**경력**: 5년~100년
**지역**: 서울
**카테고리**: IT_개발 / 프론트엔드
**마감**: 확인 못함

## 요약 (zighang summary)

### 담당업무

- 핵심 서비스 프론트 엔드 개발
- 모빌리티 서비스 TAP !(앱 / 웹), 관제 시스템 Pleos Fleet, 운송 관리 시스템 Capora 등 UMOS ONE의 핵심 서비스 프론트 엔드 개발 및 운영
- React 및 TypeScript를 주력으로 활용한 확장성과 유지 보수 용이한 웹/앱 기반 UI 개발
- 실시간 모빌리티 데이터(지도, 차량 위치, 운행 정보 등)를 효과적으로 시각화하고 안정적으로 업데이트하는 고성능 컴포넌트 개발
- UX/UI 디자이너, 백엔드 엔지니어, 서비스 기획자와 긴밀하게 협업하여 사용자 친화적 서비스 설계 및 구현
- 웹 성능 최적화, 접근성 개선 등 사용자 경험 향상을 위한 기술적 노력 주도
- 서비스 공통 디자인 시스템 구축 및 유지 보수
- E2E 테스트, 단위 테스트 등 테스트 자동화 환경 구축 및 CI/CD 파이프라인 통합
- 코드 리뷰, 아키텍처 개선 등을 통해 코드 품질 관리 및 최신 프론트 엔드 기술 트렌드 적용


### 자격요건

- React 기반 서비스 개발 및 복잡한 상태 관리(Redux, Recoil, Zustand 등) 경험이 풍부한 분
- TypeScript를 활용한 정적 타입 기반 개발에 능숙하며 견고한 코드 작성 경험
- HTML5, CSS3, JavaScript(ES6+)에 대한 깊은 이해 및 내부 동작 원리 활용 능력
- Git을 활용한 협업 및 코드 리뷰 프로세스에 익숙한 분


### 우대사항

- Next.js, Remix 등 Server Side Rendering(SSR) 또는 Static Site Generation(SSG) 기반 개발 경험
- 모바일 웹 또는 WebView 환경 서비스 개발 경험
- 대규모 서비스에서 디자인 시스템 설계 및 운영 경험
- 지도 관련 라이브러리(Mapbox, Google Maps API 등)를 활용한 지도 기반 서비스 개발 경험
- E2E 테스트 도구(Cypress, Playwright 등) 구축 및 활용 경험
- 모노 레포(Monorepo) 환경에서 공통 패키지 개발 및 관리 경험


### 전형절차

- 서류 전형
- 코딩 테스트
- 화상 면접(1시간 내외)
- 대면 혹은 화상 면접(3시간 내외)
- 최종 합격
