# Frontend Developer (보훈특별채용)

**회사**: 토스증권
**경력**: 0년
**지역**: 기타
**카테고리**: IT_개발 / 프론트엔드
**마감**: 채용시마감

## 요약 (zighang summary)

### 주요업무

- Android, iOS 앱의 웹뷰에서 동작하는 모바일 제품과 PC 브라우저에서 동작하는 데스크탑 제품, 운영을 위한 백오피스까지 모두 모던 웹 기술로 구현하고 있다
- 수많은 실시간 데이터를 끊김없이 보여주고 거래할 수 있도록 최적의 엔지니어링 기술을 활용하고 있다
- React, Next.js, Typescript 기반의 컴포넌트 아키텍처로 UI를 효율적으로 구축하고 정적 타입 검사를 통해 버그를 줄이고 있다
- React-Query, Recoil, Jotai를 사용해 서버와 클라이언트 상태를 관리하고 있다
- Emotion, Vanilla-extract를 사용해 모바일 제품과 데스크탑 제품의 스타일링을 각각 적용하고 있다
- TDS(Toss Design System)로 개발 생산성을 높이고 버그 제보부터 컴포넌트 제작까지 기여하고 있다
- 웹소켓, SSE를 활용해 실시간 데이터를 효율적으로 처리하고 서버 요청을 최적화하고 있다
- YarnBerry, PNPM로 패키지 관리를 하고 Webpack, SWC, Babel로 빠른 빌드와 배포를 구현하고 있다
- 소스 코드는 Github으로 관리되며 Github Action을 통해 자동화된 정적 분석과 테스트 리포트를 제공한다


### 자격요건

- 웹 기반의 서비스를 운영한 경험이 있는 지원자
- 투자에 관심이 있고 개발 역량 성장을 지속적으로 추구하는 분
- 예측가능한 코드의 중요성을 공감하고 이를 실행에 옮길 수 있는 분
- 변경에 유연한 아키텍처를 고민하고 구현해 본 경험이 있는 분
- 브라우저 렌더링 성능을 고려하여 개발하고 최적화한 경험이 있는 분
- 빠르고 안전하게 배포할 수 있는 서비스 구조를 고민하는 분
- 어드민, 백오피스 제품을 플랫폼화해 개발 효율을 높이고자 하는 의지가 있는 분


### 근무환경

- 차트, 그래프, 테이블 등 다양한 데이터 시각화를 경험할 수 있다
- 실시간 데이터의 시각화 성능 최적화를 접할 수 있다
- 최대 1400 TPS의 실시간 시세 트래픽을 소켓으로 다루는 경험을 할 수 있다
- 2주마다 엔지니어링 데이가 열려 기술 토픽이나 개발 노하우를 공유한다
- 생산성을 향상시키는 노하우와 새로 얻은 지식, 실수에서 배운 점을 적극 공유하는 문화가 있다
- 다른 계열사의 프론트 챕터와도 적극적으로 소통하며 공유하는 문화를 체험할 수 있다
- 토스증권 FE 챕터의 독자적 디자인 시스템을 경험하는 기회가 있다
