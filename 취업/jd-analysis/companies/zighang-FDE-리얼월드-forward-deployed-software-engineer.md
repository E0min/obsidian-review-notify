### 소개

[회사 소개]
리얼월드(RLWRLD)는 로봇이 인간처럼 세상을 인식하고 사고하며 손을 써서 작업할 수 있도록, 로보틱스 파운데이션 모델(Robotics Foundation Model)을 개발하고 있는 대한민국 최고의 AI 및 로보틱스 연구진이 이끄는 Physical AI 선도 기업입니다. 

AI와 로보틱스 분야에서 축적한 깊은 연구 역량을 바탕으로, 고자유도 로봇 손의 섬세한 조작을 가능케 하는 범용 AI 모델을 빠르게 발전시키고 있고, 또한 연구소 및 로봇·센서 솔루션 파트너들과의 강력한 네트워크를 통해 제조업 등 실제 산업 현장에서 즉각적인 시장 적용이 가능한 AI 솔루션을 개발하고 있습니다. 

최근 국내외 유수의 VC와 대기업으로부터 약 210억 원 규모의 시드 투자를 유치하며 대한민국 스타트업 역사상 최대 규모의 시드 투자를 기록한 리얼월드에는 AI와 로보틱스의 최전선 글로벌 기술 혁신을 경험하고자 하는 뛰어난 인재들이 속속 합류하고 있습니다.

[직무 소개]
Forward Deployed Software Engineer(FDSW)는 리얼월드의 RFM(Robotics Foundation Model) 기반 로봇 시스템이 산업 환경에서 실제 운영될 수 있도록, 데이터 수집·관리 파이프라인, 원격 제어(tele-op), Edge/Cloud 통합, Software 안정화를 직접 구현하는 역할입니다.

이 역할은 현장 환경을 고려한 SW Integration과 안정적 운영 아키텍처 구축을 통해 RFM이 실제 산업 현장에서 작동하도록 만드는 Software Deployment 엔지니어입니다.

FDSW는 Robotics Deployment(RD) 팀의 일원으로서, 프로젝트 관리자 및 Robotics/AI 팀과 함께 PoC → 데이터 수집 → 기술확장 → Production(실제 운영/상용화) 단계까지 End-to-End Software Delivery를 주도합니다.



### 주요업무

- 현장 데이터 수집 및 Edge/Cloud 기반 Software Setup/Integration
- Tele-operation 및 Robot-System Software 연동 구성
- 데이터 파이프라인 구축 및 현장 환경에 맞춘 안정화·최적화
- PoC 및 데이터 수집 과정에서 발생하는 SW 기술 이슈의 즉각적인 현장 대응, 원인 분석 및 해결
- 센서/시스템 로그 분석 및 디버깅을 통한 로봇 및 데이터 수집 시스템 안정화 및 최적화
- 고객/파트너 대상 운영 인프라 구축 및 기술 교육 지원
- 현장 요구사항을 기술적으로 구조화하고 내부팀에 피드백
- 불확실한 산업 환경을 SW 기반으로 해결 가능한 형태로 전환



### 자격요건

- Software/Robotics/Automation 기반 산업 현장 적용 혹은 SI/Integration 경험 *(단순 유지보수가 아닌 실제 운영 단계 적용 경험 중심)*
- Edge/Cloud 기반 Software 연동 및 Diagnostics 경험
- ROS/ROS2, gRPC, REST API, Docker 등 SW Stack 이해
- Linux 기반 시스템 운영, 로그 분석, SW Debugging 경험
- 고객사, SI 파트너와의 기술 커뮤니케이션 경험
- 다양한 환경에서 문제를 구조화하고 실행까지 연결한 경험 *(Adaptive Execution)*



### 우대사항

- Edge/Cloud SW integration 및 DevOps 경험
- Python/C++ 기반 Software 개발 경험
- Sensors/Perception/Networking 환경 이해
- ML/AI 기반 Robotics 환경 경험
- 산업 자동화/물류/제조 현장 Software 적용 경험
- 영어/일본어 기술 커뮤니케이션 능력
- Data→Model→Operation End-to-End 경험 또는 관심
