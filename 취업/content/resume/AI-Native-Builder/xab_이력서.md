### xAB (A/B 테스트 기반 SNS)

서비스 개요: 두 가지 선택지에 대한 실시간 투표 및 댓글 토론이 가능한 소셜 네트워크 서비스

팀원: 4명 (FE 4명)

기술 스택: TypeScript, Next.js, React Query, Zustand, Supabase, Tailwind CSS

- 4인 FE 협업 환경에서 인증 경계가 흐트러지지 않도록 `@supabase/ssr` 기반 Next.js 미들웨어를 도입하여 매 요청마다 서버에서 세션을 검증하고 미인증 사용자를 로그인 페이지로 리다이렉트하는 인증 게이트를 미들웨어 계층에 구현했습니다.

- 로드밸런서 환경에서 OAuth 콜백 오리진이 잘못 구성되는 운영 버그를 재발시키지 않도록 콜백 라우트에서 `X-Forwarded-Host` 헤더를 우선 참조하여 원본 오리진을 복원하는 분기를 구현하고, 분기 조건과 의도를 코드 주석으로 남겼습니다.

- 알림과 세션 정보의 전역 상태 관리를 위해 Zustand 스토어를 도입하고 Supabase Realtime 구독 핸들을 함께 보관하도록 설계하여, 컴포넌트 간 결합도를 낮추고 중복 구독으로 인한 리소스 누수를 명시적 핸들 관리로 차단했습니다.

- 실시간 댓글과 투표 상호작용이 깨지지 않도록 Supabase Realtime의 `postgres_changes` 이벤트 구독을 도입하여 서버 데이터 변경 이벤트와 클라이언트 상태를 단일 구독 경로로 통합했고, 이벤트 핸들러를 한 곳에서 관리하도록 정리했습니다.

- 게시글 상세 화면 진입 시 피드 맥락이 끊기는 문제를 해결하기 위해 Next.js App Router의 병렬 라우트와 인터셉팅 라우트를 결합한 모달 라우팅을 구현하여, 라우트 분기 책임을 App Router 규약 안에 가두었습니다.

- 메인 피드의 초기 로딩 지연 문제를 해결하기 위해 TanStack Query의 `useInfiniteQuery`와 Intersection Observer 기반 트리거를 결합하여 페이지 단위 분할 호출과 지연 로딩이 적용된 무한 스크롤 피드를 구현했습니다.
