# 구조분해할당 (Destructuring Assignment)

> TL;DR: 배열은 순서 기준, 객체는 키 이름 기준으로 값을 꺼내 변수에 한 번에 할당하는 ES6 문법이다.

## What

배열이나 객체의 값을 개별 변수에 한 줄로 추출하는 문법. 코드 가독성을 크게 높인다.

## Why it matters

함수 매개변수, API 응답 파싱, React props 처리 등 실무 전반에서 매우 빈번하게 쓰인다.

## How

### 배열 구조분해 — 순서 기반

```javascript
const arr = [10, 20, 30];

// 기본
const [first, second, third] = arr;
console.log(first, second, third); // 10 20 30

// 중간 요소 건너뛰기
const [a, , c] = [1, 2, 3];
console.log(a, c); // 1 3

// 기본값 설정
const [x = 0, y = 0] = [undefined, 5];
console.log(x, y); // 0 5
```

### 객체 구조분해 — 키 이름 기반

```javascript
const person = { name: '영민', age: 28, city: 'Seoul' };

// 기본
const { name, age } = person;
console.log(name, age); // 영민 28

// 변수명 변경 (콜론으로 별칭 지정)
const { name: userName, age: userAge } = person;
console.log(userName, userAge); // 영민 28

// 기본값 설정
const { name: n, job = '개발자' } = person;
console.log(n, job); // 영민 개발자
```

### 중첩 구조분해

```javascript
const user = {
    name: '영민',
    info: {
        age: 28,
        address: { city: 'Seoul', zip: '12345' }
    }
};

const { info: { age, address: { city } } } = user;
console.log(age, city); // 28 Seoul
```

### 함수 매개변수 구조분해

```javascript
// 객체를 그대로 받는 대신 필요한 속성만 추출
function printUser({ name, age = 0 }) {
    console.log(`이름: ${name}, 나이: ${age}`);
}

printUser({ name: '영민', age: 28 }); // 이름: 영민, 나이: 28
printUser({ name: '지수' });          // 이름: 지수, 나이: 0
```

### Rest와 함께 사용

```javascript
const { name, ...rest } = { name: '영민', age: 28, city: 'Seoul' };
console.log(name); // '영민'
console.log(rest); // { age: 28, city: 'Seoul' }
```

## Pitfalls

- 객체 구조분해 시 존재하지 않는 키를 추출하면 `undefined`가 된다. 기본값을 설정하지 않으면 예상치 못한 버그가 생긴다.
- 중첩 구조분해는 중간 단계의 속성이 `null`/`undefined`이면 TypeError가 발생한다. 옵셔널 체이닝(`?.`)과 함께 쓰는 것을 고려하라.

```javascript
const data = { user: null };

// TypeError 발생
const { user: { name } } = data;

// 안전한 방법: 기본값으로 빈 객체 제공
const { user: { name } = {} } = data;
```
