# 병렬 라우트

> TL;DR: `@슬롯명` 폴더 구조로 같은 레이아웃에서 여러 페이지를 동시에 또는 조건부로 렌더링한다.

## What

병렬 라우트(Parallel Routes)는 하나의 레이아웃에서 여러 페이지를 동시에 또는 조건에 따라 렌더링할 수 있는 App Router 기능이다. `@폴더명` 형식의 슬롯(Slot)으로 정의하며, URL 경로에는 영향을 주지 않는다.

## Why it matters

대시보드처럼 여러 독립적인 섹션을 한 화면에 보여줘야 하거나, 모달처럼 기존 레이아웃을 유지하면서 새로운 UI를 오버레이해야 할 때 유용하다. 각 슬롯은 독립적으로 로딩·에러 상태를 가질 수 있다.

## How

### 파일 구조

```
app/
├── layout.js          ← 슬롯을 props로 받는 레이아웃
├── page.js            ← children
├── @team/
│   └── page.js        ← team 슬롯
└── @analytics/
    └── page.js        ← analytics 슬롯
```

### layout.js에서 슬롯 받기

```javascript
// app/layout.js
export default function Layout({ children, team, analytics }) {
  return (
    <div>
      <aside>{team}</aside>
      <main>{children}</main>
      <section>{analytics}</section>
    </div>
  );
}
```

### 조건부 렌더링

사용자 역할에 따라 다른 슬롯을 보여줄 수 있다.

```javascript
// app/layout.js
import { checkUserRole } from "@/lib/auth";

export default async function Layout({ user, admin }) {
  const role = await checkUserRole();
  return (
    <div>
      {role === "admin" ? admin : user}
    </div>
  );
}
```

### 모달 구현 (인터셉트 라우트와 조합)

```
app/
├── layout.js
├── @modal/
│   ├── default.js     ← 모달 없을 때 null 반환
│   └── (.)photo/[id]/
│       └── page.js    ← 모달로 표시될 사진 상세
└── photo/
    └── [id]/
        └── page.js    ← 직접 접근 시 전체 페이지
```

```javascript
// app/@modal/default.js
export default function Default() {
  return null; // 슬롯 비활성 시 빈 화면
}
```

### useSelectedLayoutSegment로 활성 세그먼트 확인

```typescript
"use client";
import { useSelectedLayoutSegment } from "next/navigation";

export default function NavItem({ parallelRoutesKey }) {
  const segment = useSelectedLayoutSegment(parallelRoutesKey);
  // segment가 null이면 해당 슬롯이 비활성 상태
  return <li className={segment ? "active" : ""}>{parallelRoutesKey}</li>;
}
```

## Pitfalls

- 슬롯은 URL 경로에 포함되지 않는다. `@team`은 `/team`으로 접근할 수 없다.
- 각 슬롯에 `default.js`가 없으면 하드 내비게이션 시 404가 발생할 수 있다. 반드시 `default.js`를 추가한다.
- 슬롯 수가 많아지면 레이아웃 props가 복잡해진다. 필요한 경우에만 사용한다.
- 각 슬롯은 독립적인 `loading.js`와 `error.js`를 가질 수 있다. 활용하면 세밀한 로딩 UI 제어가 가능하다.
