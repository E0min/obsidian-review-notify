# 🌓 벨만-포드 알고리즘

> TL;DR: 단일 출발점 최단 경로 (SSSP). **음수 가중치 가능**, 음수 사이클 탐지. 모든 간선을 V-1회 반복 이완. O(VE). 다익스트라보다 느리지만 음수 간선 처리 가능.

---

# 1. 정의

벨만-포드(Bellman-Ford) 알고리즘은 **단일 출발점 최단 경로 문제(SSSP)**를 해결하는 알고리즘이다. **가중치가 음수인 간선이 존재하는 경우**에도 사용할 수 있으며, **음수 사이클** 여부도 탐지한다.

## 특징

| 항목 | 내용 |
|------|------|
| **음수 가중치** | ✅ 가능 |
| **음수 사이클 탐지** | ✅ 가능 |
| **알고리즘 패턴** | 동적 프로그래밍 (DP) |
| **시간 복잡도** | **O(V × E)** |
| **공간 복잡도** | O(V) |
| **다익스트라 대비** | 느리지만 음수 간선 처리 가능 |

---

# 2. 알고리즘 개요

- 모든 간선에 대해 **V-1회** 반복하며 최단 경로를 갱신 (이완, Relaxation)
- V-1번이면 모든 경로가 최적화되는 이유: 최단 경로는 최대 V-1개의 간선 사용
- 마지막으로 한 번 더 모든 간선 확인 → 갱신이 일어나면 **음수 사이클 존재**

---

# 3. 의사코드

```plaintext
function BellmanFord(G, source):
    for each vertex v:
        distance[v] = ∞
        predecessor[v] = null
    distance[source] = 0

    // V-1회 반복
    for i from 1 to |V| - 1:
        for each edge (u, v, w):
            if distance[u] + w < distance[v]:
                distance[v] = distance[u] + w
                predecessor[v] = u

    // 음수 사이클 탐지
    for each edge (u, v, w):
        if distance[u] + w < distance[v]:
            error "음수 사이클 존재"

    return distance[], predecessor[]
```

# 4. 자바스크립트 구현

```javascript
function bellmanFord(vertices, edges, start) {
    const distance = new Array(vertices).fill(Infinity);
    distance[start] = 0;

    // V-1번 반복하면서 모든 간선 이완
    for (let i = 1; i < vertices; i++) {
        edges.forEach(([u, v, w]) => {
            if (distance[u] !== Infinity && distance[u] + w < distance[v]) {
                distance[v] = distance[u] + w;
            }
        });
    }

    // 음수 사이클 탐지: 한 번 더 확인
    for (const [u, v, w] of edges) {
        if (distance[u] !== Infinity && distance[u] + w < distance[v]) {
            console.log("음수 사이클 탐지!");
            return null;
        }
    }

    return distance;
}

// 사용 예시
const vertices = 5;
const edges = [
    [0, 1, -1], [0, 2, 4],
    [1, 2, 3], [1, 3, 2], [1, 4, 2],
    [3, 2, 5], [3, 1, 1],
    [4, 3, -3]
];
const start = 0;

console.log(bellmanFord(vertices, edges, start));
// [0, -1, 2, -2, 1]
```

---

# 5. 동작 과정 상세

1. **초기화**: `dist = [∞, ∞, ...]`, `dist[start] = 0`
2. **V-1회 반복 이완**:
   - 첫 번째 반복: 시작 노드와 직접 연결된 노드 거리 갱신
   - 두 번째 반복: 두 단계 떨어진 노드 거리 갱신
   - k번째 반복: k단계 떨어진 노드까지 갱신
3. **음수 사이클 탐지**: V-1회 후에도 갱신이 일어나면 음수 사이클 존재

---

# 6. 다익스트라 vs 벨만-포드

| 항목 | 다익스트라 | 벨만-포드 |
|------|---------|---------|
| **음수 가중치** | ❌ | ✅ |
| **음수 사이클 탐지** | ❌ | ✅ |
| **시간 복잡도** | O(E log V) | **O(VE)** (느림) |
| **알고리즘 유형** | 그리디 | **동적 프로그래밍** |
| **적합 상황** | 양수 가중치 그래프 | 음수 가중치 포함 그래프 |

**음수 가중치가 없으면 → 다익스트라. 음수 가중치가 있으면 → 벨만-포드.**
