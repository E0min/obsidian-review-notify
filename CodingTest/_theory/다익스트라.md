# 🛕 다익스트라 알고리즘

> TL;DR: 단일 출발점 최단 경로 (Single Source Shortest Paths). **음수 가중치 불가**, 우선순위 큐 기반 그리디. O(E log V). 네트워크/지도 경로 탐색의 표준.

---

# 1. 정의

다익스트라(Dijkstra) 알고리즘은 **단일 출발점 최단 경로 문제(SSSP)**를 해결하는 알고리즘이다. **음수 가중치가 없는** 그래프에서 출발점으로부터 다른 모든 정점까지의 **최단 경로**를 구한다.

## 특징

| 항목 | 내용 |
|------|------|
| **음수 가중치** | ❌ 불가 (음수 있으면 [[벨만-포드]] 사용) |
| **자료구조** | 우선순위 큐 (최소 힙) |
| **알고리즘 패턴** | 탐욕적 알고리즘 (Greedy) |
| **시간 복잡도** | O(E log V) |
| **공간 복잡도** | O(V) |
| **적용 대상** | 유향/무향 가중치 그래프 |

---

# 2. 알고리즘 개요

1. 모든 정점까지의 거리를 `∞`로 초기화, 출발점은 0
2. 우선순위 큐에 `(0, 출발점)` 삽입
3. 큐에서 최소 거리 정점 추출 → 인접 정점 거리 갱신
4. 더 짧은 경로 발견 시 큐에 추가
5. 큐가 빌 때까지 반복

---

# 3. 의사코드

```plaintext
function Dijkstra(Graph, source):
    for each vertex v:
        dist[v] = ∞
        prev[v] = UNDEFINED
    dist[source] = 0

    PQ = MinPriorityQueue()
    PQ.add(source, 0)

    while PQ is not empty:
        u = PQ.extractMin()
        for each neighbor v of u:
            alt = dist[u] + weight(u, v)
            if alt < dist[v]:
                dist[v] = alt
                prev[v] = u
                PQ.add(v, dist[v])

    return dist[], prev[]
```

# 4. 자바스크립트 구현

```javascript
class PriorityQueue {
    constructor() {
        this.values = [];
    }

    enqueue(val, priority) {
        this.values.push({ val, priority });
        this.sort();
    }

    dequeue() {
        return this.values.shift();
    }

    sort() {
        this.values.sort((a, b) => a.priority - b.priority);
    }

    isEmpty() {
        return this.values.length === 0;
    }
}

function dijkstra(graph, start) {
    const distances = {};
    const previous = {};
    const pq = new PriorityQueue();
    const visited = new Set();

    // 초기화: 모든 정점 거리 = Infinity
    for (let node in graph) {
        distances[node] = Infinity;
        previous[node] = null;
    }
    distances[start] = 0;
    pq.enqueue(start, 0);

    while (!pq.isEmpty()) {
        const { val: currentNode } = pq.dequeue();
        if (visited.has(currentNode)) continue;
        visited.add(currentNode);

        for (let neighbor in graph[currentNode]) {
            const newDist = distances[currentNode] + graph[currentNode][neighbor];
            if (newDist < distances[neighbor]) {
                distances[neighbor] = newDist;
                previous[neighbor] = currentNode;
                pq.enqueue(neighbor, newDist);
            }
        }
    }

    return { distances, previous };
}

// 사용 예시
const graph = {
    A: { B: 1, C: 4 },
    B: { A: 1, C: 2, D: 5 },
    C: { A: 4, B: 2, D: 1 },
    D: { B: 5, C: 1 }
};

const result = dijkstra(graph, 'A');
console.log(result.distances); // { A: 0, B: 1, C: 3, D: 4 }
console.log(result.previous);  // { A: null, B: 'A', C: 'B', D: 'C' }
```

---

# 5. 동작 과정

1. **초기화**: 거리 배열 `dist` = `[∞, ∞, ...]`, `dist[start] = 0`, 우선순위 큐에 `(start, 0)` 삽입
2. **최단 거리 갱신**: 큐에서 가장 작은 거리 정점 추출 → 인접 정점 거리 갱신 → 갱신됐으면 큐에 재삽입
3. **종료**: 큐가 빌 때 모든 최단 경로 확정
