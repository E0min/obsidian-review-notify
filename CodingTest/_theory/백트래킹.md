# ⛰️ 백트래킹

> TL;DR: 체계적인 탐색으로 모든 가능한 해 찾기. 조건 위반 시 즉시 포기(pruning)하고 이전 단계로 되돌아감. DFS + 조건 검사 + 재귀. N-Queen, 부분집합, 미로 등.

- 백트래킹은 **체계적인 탐색**을 통해 모든 가능한 해를 찾아내는 알고리즘 설계 기법
- 그리디 알고리즘과 달리 모든 경우를 탐색하므로 **최적의 해를 보장**
- 주로 조합, 순열, 그래프 탐색 문제 등에서 사용되며, 해 공간을 효율적으로 탐색하기 위해 조건을 만족하지 않는 경로를 미리 포기
- 코딩테스트에서는 제한된 범위 내에서 가능한 모든 해를 탐색해야 하는 문제에 사용
- 재귀와 깊은 관련이 있으며, 재귀 호출을 통해 문제를 단계별로 해결

## 1. 백트래킹이란?

**정의**: 가능한 모든 해결책을 탐색하면서, 현재 경로가 문제의 조건을 만족하지 않으면 그 경로를 포기하고 이전 단계로 돌아가는 알고리즘 설계 기법. 탐색 공간을 줄이고 효율적으로 문제 해결.

**일반적인 특징**
- **완전 탐색**: 모든 가능한 해를 탐색
- **조건 검사**: 현재 상태가 조건을 만족하지 않으면 경로 중단 후 다른 경로 탐색
- **재귀적 접근**: 단계별로 해결하면서 재귀적으로 진행
- **스택 사용**: 호출 스택으로 경로 관리
- **효율성**: 불필요한 경로를 미리 포기 (pruning)

```java
Algorithm Backtracking(S, C)
{
    if (Solution(S))
        Record(S)
    else
        for each candidate x in C
            if (Feasible(S, x))
                Add x to S
                Remove x from C
                Backtrack(S, C)
                Add x back to C
}
```

- `Solution(S)`: 현재 집합 S가 해결책인지 확인
- `Feasible(S, x)`: 후보 x를 집합 S에 추가하는 것이 유망한지 판단
- `Add`/`Remove`: 집합 S와 후보 C 업데이트

## 백트래킹 알고리즘 사례

### 1. N-퀸 문제
N×N 체스판에 N개의 퀸을 서로 공격하지 않도록 배치.

```javascript
function solveNQueens(n) {
    const solutions = [];
    const board = Array.from({ length: n }, () => Array(n).fill('.'));

    function isSafe(row, col) {
        // 같은 열
        for (let i = 0; i < row; i++) {
            if (board[i][col] === 'Q') return false;
        }
        // 왼쪽 위 대각선
        let i = row - 1, j = col - 1;
        while (i >= 0 && j >= 0) {
            if (board[i][j] === 'Q') return false;
            i--; j--;
        }
        // 오른쪽 위 대각선
        i = row - 1; j = col + 1;
        while (i >= 0 && j < n) {
            if (board[i][j] === 'Q') return false;
            i--; j++;
        }
        return true;
    }

    function backtrack(row) {
        if (row === n) {
            solutions.push(board.map(row => row.join('')));
            return;
        }
        for (let col = 0; col < n; col++) {
            if (isSafe(row, col)) {
                board[row][col] = 'Q';
                backtrack(row + 1);
                board[row][col] = '.';  // 백트래킹
            }
        }
    }

    backtrack(0);
    return solutions;
}

console.log(solveNQueens(4));
// [[".Q..", "...Q", "Q...", "..Q."], ["..Q.", "Q...", "...Q", ".Q.."]]
```

### 2. 부분 집합 생성

```javascript
function generateSubsets(nums) {
    const subsets = [];

    function backtrack(start, currentSubset) {
        subsets.push([...currentSubset]);
        for (let i = start; i < nums.length; i++) {
            currentSubset.push(nums[i]);
            backtrack(i + 1, currentSubset);
            currentSubset.pop();  // 백트래킹
        }
    }

    backtrack(0, []);
    return subsets;
}

console.log(generateSubsets([1, 2, 3]));
// [[], [1], [1,2], [1,2,3], [1,3], [2], [2,3], [3]]
```

### 3. 미로 찾기

```javascript
function solveMaze(maze, start, end) {
    const path = [];
    const visited = Array.from({ length: maze.length }, () => Array(maze[0].length).fill(false));
    const directions = [[0, 1], [1, 0], [0, -1], [-1, 0]];

    function backtrack(x, y) {
        if (x === end[0] && y === end[1]) {
            path.push([x, y]);
            return true;
        }
        if (x < 0 || x >= maze.length || y < 0 || y >= maze[0].length || maze[x][y] === 1 || visited[x][y]) {
            return false;
        }
        visited[x][y] = true;
        path.push([x, y]);
        for (const [dx, dy] of directions) {
            if (backtrack(x + dx, y + dy)) return true;
        }
        path.pop();  // 백트래킹
        return false;
    }

    return backtrack(start[0], start[1]) ? path : null;
}
```
