# 🔮 스택과 큐

> TL;DR: 스택은 LIFO(push/pop), 큐는 FIFO(enqueue/dequeue). JS에서 스택은 배열 push+pop, 큐는 배열 push+shift. DFS는 스택, BFS는 큐.

---

# 1. 스택(Stack)

**LIFO (Last In First Out)** — 마지막에 들어간 데이터가 가장 먼저 나온다. 접시를 쌓는 것과 같음.

## 주요 연산

| 연산 | 설명 | 시간복잡도 |
|------|------|-----------|
| `push(item)` | 맨 위에 데이터 추가 | O(1) |
| `pop()` | 맨 위 데이터 제거 및 반환 | O(1) |
| `peek()` | 맨 위 데이터 반환 (제거 X) | O(1) |
| `isEmpty()` | 비어있는지 확인 | O(1) |

## 자바스크립트 구현

```javascript
class Stack {
    constructor() {
        this.items = [];
    }

    push(item) {
        this.items.push(item);
    }

    pop() {
        if (this.isEmpty()) return "스택이 비었다.";
        return this.items.pop();
    }

    peek() {
        if (this.isEmpty()) return "스택이 비었다.";
        return this.items[this.items.length - 1];
    }

    isEmpty() {
        return this.items.length === 0;
    }

    size() {
        return this.items.length;
    }

    print() {
        console.log(this.items.join(', '));
    }
}

const stack = new Stack();
stack.push(1);
stack.push(2);
stack.push(3);
stack.print();           // 1, 2, 3
console.log(stack.pop()); // 3
console.log(stack.peek()); // 2
```

# 2. 큐(Queue)

**FIFO (First In First Out)** — 먼저 들어간 데이터가 가장 먼저 나온다. 줄 서기와 같음.

## 주요 연산

| 연산 | 설명 | 시간복잡도 |
|------|------|-----------|
| `enqueue(item)` | 끝에 데이터 추가 | O(1) |
| `dequeue()` | 맨 앞 데이터 제거 및 반환 | O(n)* |
| `peek()` | 맨 앞 데이터 반환 (제거 X) | O(1) |
| `isEmpty()` | 비어있는지 확인 | O(1) |

> *JS 배열의 `shift()`는 O(n). 성능이 중요하면 연결 리스트 기반 큐 또는 인덱스 포인터 방식 사용.

## 자바스크립트 구현

```javascript
class Queue {
    constructor() {
        this.items = [];
    }

    enqueue(item) {
        this.items.push(item);
    }

    dequeue() {
        if (this.isEmpty()) return "큐가 비었다.";
        return this.items.shift();
    }

    peek() {
        if (this.isEmpty()) return "큐가 비었다.";
        return this.items[0];
    }

    isEmpty() {
        return this.items.length === 0;
    }

    size() {
        return this.items.length;
    }

    print() {
        console.log(this.items.join(', '));
    }
}

const queue = new Queue();
queue.enqueue(1);
queue.enqueue(2);
queue.enqueue(3);
queue.print();              // 1, 2, 3
console.log(queue.dequeue()); // 1
console.log(queue.peek());    // 2
```

---

# 3. JS에서 빠른 사용법

```javascript
// 스택: push + pop
const stack = [];
stack.push(1); stack.push(2); stack.push(3);
stack.pop(); // 3

// 큐: push + shift
const queue = [];
queue.push(1); queue.push(2); queue.push(3);
queue.shift(); // 1
```

---

# 4. 알고리즘과의 관계

| 알고리즘 | 사용 자료구조 | 이유 |
|---------|-------------|------|
| DFS | **스택** (또는 재귀 콜 스택) | 깊이 우선 탐색 → LIFO |
| BFS | **큐** | 너비 우선 탐색 → FIFO |
| 백트래킹 | **스택** (재귀) | 되돌아가기 → LIFO |
| 우선순위 큐 | **힙** | 우선순위 순서 → 힙 기반 |
